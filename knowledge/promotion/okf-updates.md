# Boundary

Attempts do not update OKF.
Verifier logs do not update OKF.
Passing runs do not update OKF by themselves.

Only promoted patterns update OKF.

# Checklist

Before changing this bundle:

1. Did the pattern survive a verifier?
2. Is it reusable across workflows?
3. Would Codex make a better decision next time from this note?

If any answer is no, keep it out.
