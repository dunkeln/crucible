# Shape

```text
task -> attempt -> verifier -> reward -> repair -> promotion
```

Crucible rewards evidence, not confidence. The verifier decides whether an attempt earns signal.

# Rule

No reward without a verifier.

If the verifier is unclear, the task is not ready.
