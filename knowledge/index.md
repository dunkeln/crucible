# Crucible Knowledge

Small OKF-style doctrine bundle for Codex.

Use this as context, not control. If a value changes execution, put it in a recipe, lockfile, verifier, or harness artifact instead.

## Concepts

* [RLVR loop](loops/rlvr.md) - The core loop Crucible wants Codex to preserve.
* [Operator, verifier, human](roles/operator-verifier-human.md) - The role split that keeps autonomy bounded.
* [Promotion updates knowledge](promotion/okf-updates.md) - The rule that keeps knowledge fresh without becoming noisy.
